## Wave-music - music player

### Technologies used

CSS, React, Redux.

### Installation and launch

> git clone <https://github.com/your-website/wave-music>

Install npm:

```bash
> npm install
```

Run project

> npm run build (production)

> npm run dev (development)

> npm run deploy (deploy GitHubPages)
